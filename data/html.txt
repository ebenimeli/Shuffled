Meta tag - Provides metadata about an HTML document.
Headings - Create hierarchical structure and organization.
Paragraph - Creates a block of text.
Image - Embeds images into an HTML document


ID - Identifies a unique HTML element.
Style - Applies styles directly to an HTML element.
Display - Specifies the type of box model.
Position - Positions an HTML element.
Media query - Applies styles based on device characteristics.



Radio - Creates a radio button for user selection.
Textarea - Creates a multi-line input field.
Bold - Makes text bold.
Italic - Makes text italic.
Underline - Underlines text.
Strike - Strikes through text.

Subscript - Makes text smaller and below the line.

Color - Specifies the color of text.

Border - Specifies the border of an element.
Margin - Specifies the space outside of an element.
Padding - Specifies the space inside of an element.
Display - Specifies how an element is displayed.
